# Día 3 · Sistemas dinámicos aplicados a la economía

Aplicación web desarrollada con Vite y React para la lección:

- Modelo de Evans con ajuste de precios
- Modelo de precios con ecuación diferencial de segundo orden

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Despliegue

El proyecto queda listo para desplegarse directamente en Vercel como una aplicación Vite.
